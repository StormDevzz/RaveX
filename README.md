## RU

RaveX — клиентский мод для Minecraft от организации StormDevzz.

### Сборка

```bash
./gradlew build
```

Готовый JAR находится в `build/libs/`.
Готовые сборки также доступны в разделе Releases.

### Установка

1. Установите Fabric Loader
2. Поместите JAR в папку `mods`
3. Запустите игру

## EN

RaveX is an open-source Minecraft client modification by StormDevzz organization.

### Building

```bash
./gradlew build
```

The built JAR will be in `build/libs/`.
Pre-built JARs are also available in the Releases section.

### Installation

1. Install Fabric Loader
2. Place the JAR in your `mods` folder
3. Launch the game
