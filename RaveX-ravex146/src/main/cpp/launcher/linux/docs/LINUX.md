# Linux only

Данная версия лаунчера предназначена **исключительно для Linux**.

## Почему только Linux

- Использует системные вызовы Linux: `fork()`, `execvp()`, `setpgid()`, `pipe()`, `kill()`
- Собирается под GTK3 (нативный Linux тулкит)
- Использует `popen()` с shell-командами (`curl`, `mkdir`, `cp`)
- Файловые пути соответствуют Linux FHS (`~/.config`, `~/.local/share`)

## Зависимости

Без этих пакетов лаунчер не запустится:

| Пакет         | Назначение                |
|---------------|---------------------------|
| `libgtk-3-dev`| Графический интерфейс     |
| `curl`        | HTTP-запросы              |
| `cmake`       | Сборка                    |
| `g++`/`clang` | Компиляция C++17          |

## Планы

Поддержка Windows и macOS в будущем не планируется. Для других платформ рекомендуется использовать официальный лаунчер Minecraft.
