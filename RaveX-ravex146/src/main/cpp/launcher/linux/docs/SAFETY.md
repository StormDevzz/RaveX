# Safety

## Open Source

Исходный код лаунчера полностью открыт. Вы можете проверить каждый файл перед сборкой.

## Что лаунчер делает

- Подключается к GitHub API для проверки обновлений
- Скачивает release-артефакты с GitHub (RaveX клиент)
- Скачивает версии Minecraft напрямую с Mojang (piston-meta.mojang.com)
- Использует Microsoft / offline аутентификацию для входа в Minecraft

## Безопасность

- **Данные не собираются.** Лаунчер не отправляет телеметрию, не логирует личные данные.
- **Учётные данные.** Токены и пароли не хранятся. Только username + uuid + токен для оффлайн-режима. Microsoft-сессия живёт только во время работы.
- **Сеть.** HTTP-запросы только к GitHub API, Mojang API и Microsoft Live. Никаких сторонних серверов.
- **Файловая система.** Лаунчер работает только в пределах `~/.kickxxx/`, `~/.ravex/` и `~/.minecraft/mods/`.

## Сборка из исходников

Рекомендуется всегда собирать лаунчер самостоятельно, чтобы гарантировать, что бинарник не содержит изменений.

```bash
cd src/main/cpp/launcher/linux
./build.sh
```
