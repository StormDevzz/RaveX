# RaveX

**Developer: StormDevzz** (github.com/StormDevzz)

---

RaveX is an open-source Minecraft client modification by StormDevzz organization.

### Languages in this project

| Language | Purpose |
|---|---|
| **Java** | Core mod logic — ClickGUI, modules, mixins, rendering |
| **C++** | Native code for performance-critical operations: optimizer, anti-AFK, shader hooks, JNI bridge |
| **Lua** | Scripting — rich presence, Discord integration, custom modules and Lua addons |
| **Makefile** | Generated by CMake for building C++ components (native libraries) |

### Building

```bash
./gradlew build
```

The built JAR will be in `build/libs/`.
Pre-built JARs are also available in the Releases section.

### Installation

1. Install Fabric Loader
2. Place the JAR in your `mods` folder
3. Launch the game
