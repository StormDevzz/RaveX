# RaveX C Addon Templates

Native addons for RaveX written in pure C (only Linux).
Нативные аддоны для RaveX на чистом C (только Linux).

## Quick start / Быстрый старт

```bash
chmod +x scripts/build.sh
./scripts/build.sh 01_minimal
```

## Examples / Примеры

- `01_minimal` - minimal addon with init/shutdown
- `02_features` - addon with tick and key handlers
