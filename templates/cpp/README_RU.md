# Сборка и установка C++ нативных аддонов RaveX

Этот каталог содержит шаблон C++ нативного аддона для RaveX, который демонстрирует регистрацию модулей через C++ API.

Аддоны спроектированы так, чтобы работать **только совместно с RaveX**. Без установленного клиента RaveX они функционировать не будут.

---

## Папка установки

Собранная разделяемая библиотека (`.so` для Linux, `.dll` для Windows) должна быть помещена в специальную папку `ravex/addons/native/` внутри каталога твоего инстанса (профиля) Minecraft.

| Лаунчер / ОС | Путь к папке нативных аддонов |
| :--- | :--- |
| **Обычный лаунчер** | `<Minecraft-game-directory>/ravex/addons/native/` |
| **Prism Launcher (Flatpak на Linux)** | `~/.var/app/org.prismlauncher.PrismLauncher/data/PrismLauncher/instances/<Имя_Инстанса>/minecraft/ravex/addons/native/` |

---

## Инструкция по сборке

### Требования
* Компилятор C++ с поддержкой стандарта C++17/20 (GCC, Clang, MSVC).
* Установленный CMake версии 3.10 или выше.

### Шаги сборки

1. **Создай директорию сборки и перейди в неё**:
   ```bash
   mkdir -p build
   cd build
   ```

2. **Сгенерируй файлы конфигурации сборки с помощью CMake**:
   ```bash
   cmake ..
   ```

3. **Скомпилируй библиотеку** (в конфигурации Release для оптимизации):
   ```bash
   cmake --build . --config Release
   ```
   После компиляции в каталоге `build/` появится собранный файл:
   * `AnotherAddon.so` — на Linux.
   * `AnotherAddon.dll` — на Windows.

4. **Скопируй полученную библиотеку** в папку `ravex/addons/native/` твоего инстанса Minecraft.

> Все шаги компиляции и копирования автоматизированы в скрипте [build.sh](file:///home/nprevenant/RaveX/templates/cpp/build.sh). Можешь запустить его локально.
